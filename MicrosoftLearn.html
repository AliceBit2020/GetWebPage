Поля

Имя Значение Описание
Accept 20

Заголовок Accept, указывающий типы MIME, допустимые для ответа.
AcceptCharset 21

Заголовок Accept-Charset, указывающий наборы символов, допустимые для ответа.
AcceptEncoding 22

Заголовок Accept-Encoding, который указывает кодировку содержимого, допустимую для ответов.
AcceptLanguage 23

Заголовок Accept-Language, который указывает предпочтительные естественные языки для ответов.
Allow 10

Заголовок Allow, который указывает набор поддерживаемых методов HTTP.
Authorization 24

Заголовок Authorization, который указывает учетные данные, предоставляемые клиентом для прохождения проверки подлинности на сервере.
CacheControl 0

Заголовок Cache-Control, который указывает директивы кэширования, которые должны выполняться всеми механизмами управления кэшированием по всей цепи запросов и ответов.
Connection 1

Заголовок Connection, который указывает параметры, необходимые для определенного подключения.
ContentEncoding 13

Заголовок Content-Encoding, который указывает кодировки, которые были применены к данным основного текста.
ContentLanguage 14

Заголовок Content-Language, который указывает естественный язык или языки данных в основном тексте.
ContentLength 11

Заголовок Content-Length, который указывает длину (в байтах) данных в основном тексте.
ContentLocation 15

Заголовок Content-Location, который указывает URI, с которого можно получить основной текст.
ContentMd5 16

Заголовок Content-MD5, указывающий дайджест MD5 сопроводительных данных основного текста с целью предоставления сквозной проверки целостности сообщения. Из-за проблем с MD5 Майкрософт рекомендует использовать модель безопасности на основе SHA256 или более высокого уровня.
ContentRange 17

Заголовок Content-Range, указывающий, в каком месте полного основного текста должна применяться сопроводительная часть данных основного текста.
ContentType 12

Заголовок Content-Type, который указывает тип MIME данных в основном тексте.
Cookie 25

Заголовок Cookie, который указывает данные файла cookie, предоставляемого сервером.
Date 2

Заголовок Date, который указывает дату и время создания запроса.
Expect 26

Заголовок Expect, который указывает определенное поведение сервера, требуемое клиентом.
Expires 18

Заголовок Expires, который указывает дату и время, после которого данные в основном тексте могут считаться устаревшими.
From 27

Заголовок From, указывающий адрес электронной почты в Интернете для пользователя, управляющего запрашивающим агентом пользователя.
Host 28

Заголовок Host, который указывает имя узла и номер порта запрашиваемого ресурса.
IfMatch 29

Заголовок If-Match, указывающий, что затребованная операция должна выполняться только в том случае, если копия указанного ресурса в кэше клиента является текущей.
IfModifiedSince 30

Заголовок If-Modified-Since, который указывает, что запрошенная операция должна выполняться только в том случае, если запрошенный ресурс был изменен с указанной даты и времени.
IfNoneMatch 31

Заголовок If-None-Match, указывающий, что затребованная операция должна выполняться только в том случае, если ни одна из копий указанных ресурсов в кэше клиента не является текущей.
IfRange 32

Заголовок If-Range, указывающий, что если копия в кэше клиента является текущей, то отправляться должен только указанный диапазон запрошенного ресурса.
IfUnmodifiedSince 33

Заголовок If-Unmodified-Since, который указывает, что требуемая операция должна выполняться только в том случае, если запрашиваемый ресурс не был изменен с момента указанной даты и времени.
KeepAlive 3

Заголовок Keep-Alive, который указывает параметр, используемый для поддержки постоянного подключения.
LastModified 19

Заголовок Last-Modified, который указывает дату и время последнего изменения данных в основном тексте.
MaxForwards 34

Заголовок Max-Forwards, который указывает целое число, обозначающее количество оставшихся перенаправлений этого запроса.
Pragma 4

Заголовок Pragma, который указывает директивы, относящиеся к реализации. Эти директивы можно применять к любому агенту в цепи запросов и ответов.
ProxyAuthorization 35

Заголовок Proxy-Authorization, который указывает учетные данные, предоставляемые клиентом для прохождения проверки подлинности на прокси-сервере.
Range 37

Заголовок Range, указывающий поддиапазон (или поддиапазоны) ответа, запрошенного клиентом, которые будут возвращены вместо полного ответа.
Referer 36

Заголовок Referer, который указывает URI ответа, из которого был получен URI запроса.
Te 38

Заголовок TE, который указывает кодировки передачи, допустимые для ответов.
Trailer 5

Заголовок Trailer, который определяет, что поля заголовка предоставляются в анонсе сообщения, закодированном с помощью поблочного кодирования.
TransferEncoding 6

Заголовок Transfer-Encoding, который указывает, какой тип преобразования был применен к основному тексту (если применимо).
Translate 39

Заголовок Translate — расширение Microsoft для спецификации HTTP, используемой вместе с функциональными возможностями WebDAV.
Upgrade 7

Заголовок Upgrade, который указывает дополнительные протоколы связи, поддерживаемые клиентом.
UserAgent 40

Заголовок User-Agent, который указывает сведения об агенте клиента.
Via 8

Заголовок Via, который указывает промежуточные протоколы, которые используются агентом шлюза или прокси-агентом.
Warning 9

Заголовок Warning, который указывает дополнительные сведения о состоянии или преобразовании сообщения, которое может и не быть отражено в самом сообщении.

Комментарии

Соответствующее содержимое различных заголовков подробно описано в спецификации HTTP/1.1, доступной по адресу rfc2616.
